# ShopSmart SmartRewards Management System

## Description
A C# console-based customer loyalty management system that allows businesses to manage customers and their SmartRewards accounts. The system enables users to register customers, record purchases, update customer details, remove customers, and display customer information through a simple menu-driven interface.

## Features
- Register new customers
- Record customer purchases
- Update customer details
- Remove customers from the system
- Display customer information and options
- Menu-driven user interface
- Handles invalid user input

## Technologies Used
- C#
- .NET
- Visual Studio

## Concepts Practiced
- Object-Oriented Programming (OOP)
- Classes and objects
- Methods and modular programming
- Data management
- User input handling
- Conditional statements and loops
- Console application development
